- After `Bun.spawn(command, ...)`, create a timeout race:
  ```
  const timeoutMs = options?.timeoutMs ?? 30_000;
  const GRACE_MS = 5_000;
  let timedOut = false;
  const timer = setTimeout(() => {
    timedOut = true;
    proc.kill('SIGTERM');
    setTimeout(() => {
      try { proc.kill('SIGKILL'); } catch {}
    }, GRACE_MS);
  }, timeoutMs);
  ```
- After `await proc.exited`, call `clearTimeout(timer)`
- If logger provided: `logger.info('subprocess', 'exec', { command: command.join(' '), exitCode, timedOut, durationMs: Date.now() - start })`
- Add `const start = Date.now()` before spawn
- Return `timedOut` in the result object
- If timedOut, set `ok: false`, `stderr: 'Process timed out after ${timeoutMs}ms'`
