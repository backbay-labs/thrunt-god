    Add 4 new public async methods to WorkspaceService. All methods follow the pattern: check mcpClient existence -> call mcpClient.callTool -> parse response -> use pure functions from mcp-enrichment.ts -> vault I/O.

    Add imports at top of workspace.ts:
    ```typescript
    import { mergeEnrichment, buildCoverageReport, formatDecisionEntry, formatLearningEntry } from './mcp-enrichment';
    import type { EnrichmentData, CoverageReport, CoverageTactic, McpToolResult } from './types';
    ```

    **Method 1: `async enrichFromMcp(notePath: string): Promise<{success: boolean; message: string}>`**
    - Guard: if !this.mcpClient or !this.mcpClient.isConnected(), return { success: false, message: 'MCP is not connected. Enable MCP in settings.' }
    - Read note content via `this.vaultAdapter.readFile(notePath)`
    - Parse `mitre_id` from YAML frontmatter using regex: match `/^mitre_id:\s*"?([^"\n]+)"?$/m`
    - If no mitre_id found, return { success: false, message: 'No mitre_id found in frontmatter.' }
    - Call `this.mcpClient.callTool('lookupTechnique', { id: mitre_id })`
    - If result is null or result.isError, return { success: false, message: 'MCP lookup failed.' }
    - Parse result.content[0].text as JSON into EnrichmentData shape (wrap in try/catch)
    - Call `mergeEnrichment(existingContent, enrichmentData)` to get updated content
    - Write via `this.vaultAdapter.modifyFile(notePath, updatedContent)`
    - Invalidate cache, return { success: true, message: `Enriched ${mitre_id} from MCP.` }

    **Method 2: `async analyzeCoverage(): Promise<{success: boolean; message: string}>`**
    - Guard: if !this.mcpClient or !this.mcpClient.isConnected(), return { success: false, message: 'MCP is not connected.' }
    - Resolve planning dir via getPlanningDir
    - Read all .md files from entities/ttps/ folder using vaultAdapter.listFiles
    - For each file, read content, parse `mitre_id` from frontmatter, parse `hunt_count` from frontmatter (default 0)
    - Collect technique IDs array
    - Call `this.mcpClient.callTool('analyzeCoverage', { techniqueIds })`
    - If result is null or result.isError, return { success: false, message: 'MCP coverage analysis failed.' }
    - Parse result.content[0].text as JSON into CoverageReport shape
    - Call `buildCoverageReport(report.tactics, report.totalTechniques, report.huntedTechniques, report.overallPercentage, report.gaps)`
    - Write to `{planningDir}/COVERAGE_REPORT.md` using createFile or modifyFile (overwrite if exists)
    - Invalidate cache, return { success: true, message: `Coverage report written to COVERAGE_REPORT.md` }

    **Method 3: `async logDecision(notePath: string, decision: string, rationale: string): Promise<{success: boolean; message: string}>`**
    - Guard MCP connection (same pattern)
    - Read note, parse mitre_id from frontmatter
    - Call `this.mcpClient.callTool('logDecision', { technique: mitre_id, decision, rationale })`
    - Generate local entry via `formatDecisionEntry(mitre_id, decision, rationale)`
    - Append entry to note: find `## Decisions` section, or if missing, add `\n\n## Decisions\n\n` before appending
    - Write via modifyFile
    - Invalidate, return success

    **Method 4: `async logLearning(topic: string, learning: string): Promise<{success: boolean; message: string}>`**
    - Guard MCP connection
    - Call `this.mcpClient.callTool('logLearning', { topic, learning })`
    - Generate local entry via `formatLearningEntry(topic, learning)`
    - Append to `{planningDir}/LEARNINGS.md`:
      - If file exists: read + append entry
      - If not: create with `# Hunt Learnings\n\n` header + entry
    - Invalidate, return success
