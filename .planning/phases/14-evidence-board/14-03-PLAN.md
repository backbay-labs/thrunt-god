    **Component: MatrixView**

    Props: `{ viewModel: EvidenceBoardViewModel; focusedHypothesis: string | null; onHypothesisFocus: (id: string | null) => void }`

    Implementation:

    1. **Build a cell lookup map** from `viewModel.matrixCells`:
       ```typescript
       const cellMap = useMemo(() => {
         const map = new Map<string, EvidenceBoardMatrixCell>();
         for (const cell of viewModel.matrixCells) {
           map.set(`${cell.hypothesisId}:${cell.receiptId}`, cell);
         }
         return map;
       }, [viewModel.matrixCells]);
       ```

    2. **Detect gap rows and columns:**
       - Gap row: a receiptId where ALL cells for that receipt have relationship='absent'
       - Gap column: a hypothesisId where ALL cells for that hypothesis have relationship='absent'
       ```typescript
       const gapRows = useMemo(() => {
         const gaps = new Set<string>();
         for (const rId of viewModel.receiptIds) {
           const allAbsent = viewModel.hypothesisIds.every(hId => {
             const cell = cellMap.get(`${hId}:${rId}`);
             return !cell || cell.relationship === 'absent';
           });
           if (allAbsent) gaps.add(rId);
         }
         return gaps;
       }, [viewModel, cellMap]);

       const gapCols = useMemo(() => {
         const gaps = new Set<string>();
         for (const hId of viewModel.hypothesisIds) {
           const allAbsent = viewModel.receiptIds.every(rId => {
             const cell = cellMap.get(`${hId}:${rId}`);
             return !cell || cell.relationship === 'absent';
           });
           if (allAbsent) gaps.add(hId);
         }
         return gaps;
       }, [viewModel, cellMap]);
       ```

    3. **Render table:**
       - First row: empty corner cell, then column headers (hypothesis IDs, truncated to 8 chars). Gap columns get `hunt-eb-matrix__gap-col-header` class. Clicking a column header calls `onHypothesisFocus(hypId)` if not already focused, or `onHypothesisFocus(null)` to clear.
       - Subsequent rows: row header (receipt ID), then cells. Each cell renders as:
         ```tsx
         <td>
           <div class={`hunt-eb-matrix__cell hunt-eb-matrix__cell--${cell.relationship}`}
                title={`${cell.hypothesisId} / ${cell.receiptId}: ${cell.relationship}${cell.deviationScore != null ? ` (score: ${cell.deviationScore})` : ''}`}>
             {cell.relationship === 'absent' ? '' : cell.deviationScore != null ? cell.deviationScore : ''}
           </div>
         </td>
         ```
       - Gap rows get `hunt-eb-matrix__gap-row` class on the `<tr>`. The row header shows a `<span class="hunt-eb-matrix__no-coverage">No coverage</span>` indicator next to the receipt ID.
       - When `focusedHypothesis` is set, columns not matching it get 0.15 opacity (inline style `opacity: focusedHypothesis && hId !== focusedHypothesis ? 0.15 : 1`).

    4. **Deviation score as opacity intensity on cells:**
       If `cell.deviationScore` is not null, apply opacity scaling to the cell background:
       ```typescript
       const intensity = cell.deviationScore != null
         ? 0.4 + (Math.min(cell.deviationScore, 6) / 6) * 0.6
         : 1;
       ```
       Apply as `style={{ opacity: intensity }}` on the cell div.

    5. **Blind spot callout section below the table:**
       ```tsx
       {viewModel.blindSpots.length > 0 && (
         <div class="hunt-eb-matrix__blind-spots">
           <p class="hunt-eb-matrix__blind-spots-heading">Blind Spots</p>
           {viewModel.blindSpots.map((spot, i) => (
             <div class="hunt-eb-matrix__blind-spot-item" key={i}>{spot}</div>
           ))}
         </div>
       )}
       ```

    6. **Empty state:** If `viewModel.hypothesisIds.length === 0 && viewModel.receiptIds.length === 0`, render:
       ```tsx
       <Panel>
         <p style={{ color: 'var(--hunt-text-muted)', margin: 0 }}>No evidence data to display in matrix.</p>
       </Panel>
       ```
