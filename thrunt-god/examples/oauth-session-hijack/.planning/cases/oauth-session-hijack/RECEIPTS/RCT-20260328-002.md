# Receipt: Mailbox Forwarding Rule to External Address Confirmed

## Claim

HYP-02 is supported: the attacker established email exfiltration by creating a mailbox forwarding rule in sarah.chen@acme.corp's mailbox. The rule "Auto-Forward" forwards all incoming mail to ext-recv-4782@protonmail.com. The rule was created 4 minutes after the OAuth consent grant, using the malicious app's OAuth token.

## Evidence

From QRY-20260328-002 (M365 email/alerts, templates=2):

- New-InboxRule "Auto-Forward" created by sarah.chen@acme.corp at 2026-03-28T08:46:12Z
- Rule condition: ApplyToAllMessages (forwards everything)
- Rule action: ForwardTo ext-recv-4782@protonmail.com
- ClientIP: 185.220.101.42 (same Tor exit node as OAuth consent)
- ClientApplication: 3fa85f64-5717-4562-b3fc-2c963f66afa6 (the malicious "DocuSign Secure View" app)
- Defender alert "Suspicious inbox rule" fired at 2026-03-28T08:46:18Z (6 seconds after rule creation)
